Validaciones JS

Los campos se validaran de la siguiente manera

  • Ficha: se puede ingresar cualquier caracter
  • Nombre: solo se puede ingresar letras
  • Apellidos: solo se puede ingresar letras
  • Correo: solo se puede ingresar un correo valido
  • Teléfono: solo se puede ingresar numeros
  • Genero: solo se puede seleccionar un genero
  • Ciudad: solo se puede seleccionar una ciudad
  • Habilidades: se tiene que seleccionar como minimo 3 habilidades
